## 🚀 快速开始

### 学习路径建议

1. **初学者路径**：
   - 先学习理论篇（第1-6章），建立基础知识
   - 然后跟随实战篇（第1-15章），逐步实现项目
   - 最后学习求职篇，准备面试

2. **有经验开发者路径**：
   - 快速浏览理论篇，重点关注 Agent Loop 和 RAG
   - 重点学习实战篇，关注架构设计和实现细节
   - 参考求职篇，准备项目展示

3. **面试准备路径**：
   - 重点学习理论篇和求职篇
   - 理解项目核心架构和设计思路
   - 准备常见面试题的回答

## 📋 前置知识

在学习本教程之前，建议你具备以下基础知识：

- **Java 编程**：熟悉 Java 8+ 语法和面向对象编程
- **Spring Boot**：了解 Spring Boot 基础用法
- **数据库**：熟悉 SQL 和关系型数据库
- **RESTful API**：了解 HTTP 协议和 REST API 设计
- **基础 AI 概念**：了解机器学习和大语言模型的基本概念（可选）

## 🎯 学习目标

完成本教程后，你将能够：

1. ✅ 理解 AI Agent 的核心概念和工作原理
2. ✅ 掌握 RAG 技术的实现和应用
3. ✅ 使用 Spring AI 框架构建 AI 应用
4. ✅ 实现完整的 Agent Loop 机制
5. ✅ 构建可扩展的工具系统
6. ✅ 实现知识库管理和 RAG 检索
7. ✅ 设计 RESTful API 和实时通信
8. ✅ 部署和测试 AI Agent 系统

## 📝 项目信息

- **项目名称**：JChatMind
- **技术栈**：Spring Boot 3.5.8、Spring AI、PostgreSQL、MyBatis
